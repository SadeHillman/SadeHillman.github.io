About

I chose to research more about the use of "display: inline;" vs "display: inline-block;" because it is one of the things I had difficulty understanding the most! While trying to get the layout of my web page right I noticed when using different element types they all moved differently.

What does it mean to display: inline;?

Using the display: inline; property only takes up the space of its content. This means we aren't able to add height or width attributes to style.

What does it mean to display: inline-block;?

Using display: inline-block; we can display elemennts with the inline format but now we can add height and width.

Both inline and inline-block displays have their important roles throughout our web page layouts! Inline helps us add things like links inside paragraphs and keep them tidy without having to change a lot of properties. With inline-block we can adjust the content to fit images and paragraphs on the same line.
